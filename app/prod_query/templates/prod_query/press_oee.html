Data for Press xyz:

total_potential_minutes (usually 7200mins)= x   
planned_downtime = y   
total_downtime = z   
actual_parts = a   
target_parts = b   

1. Planned Production Time (PPT):
planned_production_time = total_potential_minutes - planned_downtime

2. Run Time:
Unplanned downtime is calculated as total_downtime - planned_downtime.
Run Time is then planned_production_time - unplanned_downtime.

3. Ideal Cycle Time (minutes per part):
ideal_cycle_time = planned_production_time / target_parts

4. Availability:
availability = run_time / planned_production_time

5. Performance:
performance = (ideal_cycle_time * actual_parts) / run_time

6. Quality:
Assumed to be 100% for now, so quality = 1.0

Overall Equipment Effectiveness (OEE):
oee = availability * performance * quality
